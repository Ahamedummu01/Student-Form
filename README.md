# 🎓 Student Form with JavaScript

This is a simple **Student Management Form** created using **HTML, CSS, and JavaScript**.
Users can enter student details and display them in a table dynamically.

## 🚀 Features

* Add student details using a form
* Real-time data display in table
* Delete student record option
* Clean and simple UI design
* Form fields automatically clear after saving

## 🛠️ Technologies Used

* HTML
* CSS
* JavaScript

## 📋 Form Fields

The form collects the following information:

* Name
* Age
* Gender
* Course
* Email

After clicking **Save Details**, the entered data will appear in the **Student Details table**.

## ❌ Delete Option

Each row contains a **Delete button** that allows users to remove a student record instantly.
